# ADB Device Manager

Windows에서 실행되는 ADB 장치 관리 GUI 프로그램입니다.

## 기능

1. **장치 선택**: 상단 콤보박스에서 연결된 ADB 장치를 선택
2. **All Devices**: 모든 장치에 동시에 명령 실행
3. **JSON 설정**: 사용자 정의 명령어를 JSON 파일로 관리
4. **실시간 로그**: 명령 실행 결과를 하단에 표시

## 설치 및 실행

### 필요사항
- Python 3.7 이상
- ADB (Android Debug Bridge)가 설치되어 있고 PATH에 등록되어 있어야 함

### 실행 방법
```bash
python adb_manager.py
```

## JSON 설정 파일 (adb_commands.json)

프로그램 실행 시 자동으로 생성되며, 다음과 같은 형식을 사용합니다:

```json
{
    "commands": [
        {
            "name": "명령 이름",
            "command": "adb -s ADBID shell 명령어"
        }
    ]
}
```

- `ADBID`는 자동으로 선택된 장치 ID로 치환됩니다
- 새로운 명령을 추가하려면 JSON 파일을 수정하고 프로그램을 재시작하세요

## 사용 예시

1. 프로그램 실행
2. 상단에서 장치 선택 (또는 "all devices" 선택)
3. 원하는 명령 버튼 클릭
4. 하단 로그에서 실행 결과 확인

## 주의사항

- ADB 명령어 중 일부(예: 재부팅)는 장치에 직접적인 영향을 미칩니다
- "all devices" 선택 시 모든 연결된 장치에 명령이 실행되므로 주의하세요
- 장치가 인식되지 않으면 "새로고침" 버튼을 눌러주세요
